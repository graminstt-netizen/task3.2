===================================================================
ОПИСАНИЕ ПРОЕКТА И СТРУКТУРА
===================================================================
Консольная программа на языке C для выполнения арифметических операций
над большими числами (BigNum) с использованием компактных битовых массивов.

Особенности реализации:
* Компактное хранение элементов фиксированной битовой длины N в массиве слов.
  Биты элементов упакованы "встык", один элемент может располагаться 
  на стыке нескольких соседних слов в памяти.
* Настраиваемый при компиляции тип слова (unsigned char, unsigned int, 
  uint64_t и др.) через определение 'BitsArrayElementType'.
* Поддерживаемые битовые длины N для элементов: {4, 8, 16, 32, 64}.
* Реализованные операции:
  - Инициализация большого числа из строки шестнадцатеричных символов.
  - Сложение больших чисел (в столбик с переносом разряда).
  - Вычитание больших чисел (в столбик с заёмом разряда).
  - Форматированный вывод большого числа в консоль.

Структура проекта:
* app/ — исполняемое приложение (считывает аргументы командной строки, 
         вызывает функции вычислений и выводит результат).
* lib/ — библиотека функций (содержит низкоуровневые функции чтения/записи 
         битового массива BitsArrayGet/BitsArraySet и логику длинной арифметики).

===================================================================
ПОДРОБНАЯ ИНСТРУКЦИЯ ПО СБОРКЕ И ЗАПУСКУ В POWERSHELL
===================================================================

1. Очистка кэша и сборка проекта

# Удаляем старую папку сборки (если она была), чтобы избежать конфликтов кэша:
> Remove-Item -Recurse -Force build

# Создаем чистую папку для сборки и переходим в нее:
> mkdir build
> cd build

# Настраиваем CMake (сборка без тестов):
> cmake ..

# Запускаем компиляцию проекта:
> cmake --build . --config Debug


2. Ручной запуск программы и тестирование

После успешной сборки исполняемый файл "my_program.exe" появится в подпапке 
"build/app/Debug/". Программа принимает два шестнадцатеричных числа в качестве 
аргументов командной строки, складывает их и вычитает.

* Пример запуска (сложение и вычитание двух шестнадцатеричных чисел):
  > ./app/Debug/my_program.exe A1B2C3D4 10203040

* Пример вывода программы в консоль:
  BigNum Calculator started
  First number:  A1B2C3D4
  Second number: 10203040
  Sum:           B1D2F414
  Difference:    91929394
